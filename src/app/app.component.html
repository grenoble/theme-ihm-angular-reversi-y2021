Les étapes à suivre :
  1. Ajouter une dépendance au service de jeu Reversi dans le composant racine (AppComponent).
  2. Générez un composant plateau
  3. Placez-vous dans le répertoire du projet.
  4. Générez le composant :
    node ./node_modules/@angular/cli/bin/ng generate component plateau -c OnPush
  5. Ce composant définit 3 attribut inputs (utiliser @Input() ) :
    • board, de type Board_RO
    • turn, de type Turn
    • playable, du type tableau de tableau de booléen => Un tableau de 8x8 qui indique où le joueur courant peut jouer.
    • Initialisez ces attributs avec des valeurs par défaut représentants un tableau vide sur lequel le joueur 1 ne peut jouer nulle part.
  6. Il définit un attribut output : play, qui émet un doublet de nombres (soit un tuple [number, number]) représentant les coordonnées de la case sur laquelle le joueur veut poser son pion.
  7. Intégrez ce composant dans la vue du composant racine.
  8. Codez la vue du composant : un tableau de 8 lignes et 8 colonnes. Reprenez le principe de la vue du reversi MVP :
    1. Les cases du tableau contiennent une balise div si la case contient le pion d'un joueur.
    2. Les cases contenant un pion du joueur 1 contiennent la classe CSS "Player1".
    3. Les cases contenant un pion du joueur 2 contiennent la classe CSS "Player2".
    4. Les cases sur lesquelles le joueur courant peut poser un pion contiennent la classe CSS "canPlay.
    5. Vous pouvez réutiliser le code CSS du projet Reversi MVP et l'intégrer dans le code SCSS du composant Plateau.
    6. Donner la possibilité au joueur de poser un pion sur les case du plateau (si il en a le droit).
    7. Afficher aussi : à qui est-ce de jouer (avec le pion qui lui correspond) et le nombre totale de pions pour chaque joueur.
